# Aver Production Website

A cinematic static website for **Aver Production**, a Turkey-based independent film production company.

## Overview

This website presents Aver Production’s filmography, company profile, featured works, awards, and contact information with a dark cinematic visual style.

## Features

- Responsive landing page
- Dark cinematic design
- Featured film cards
- About Aver Production section
- Erkut Altındağ profile section
- Awards & festivals section
- Contact call-to-action
- Ready for GitHub Pages deployment
